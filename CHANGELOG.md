# Change Log

All notable changes to the "oklch-shade" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.4] - 2026-01-11

- Lower the minimum VS Code engine to ^1.104.0 for Antigravity compatibility.

## [1.0.3] - 2026-01-04

- Add marketplace icon.

## [1.0.2] - 2026-01-04

- Remove leftover packaging artifacts from the published extension.

## [1.0.1] - 2026-01-04

- Raw token previews for OKLCH/OKLab/LCH/Lab/RGB/HSL/HWB/Display P3 and hex.
- Standard CSS function parsing (oklch/oklab/lab/lch/rgb/hsl/hwb/color(display-p3)).
- Convert command with copy/replace/replace-all.
- Batch convert command for file/folder/workspace (Explorer context menu).
- Inline convert hint + status bar convert action.
- Hover previews with multiple output formats.
- `@color` and `@convert` directives, plus `@space` hints.
